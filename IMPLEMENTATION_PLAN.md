# IMPLEMENTATION_PLAN.md

# Project Overview

## Project Summary

Hasmunandar Portfolio Website adalah personal portfolio berbasis storytelling yang berfokus pada perjalanan, identitas, pertumbuhan, dan karya.

Implementasi harus mempertahankan filosofi editorial, identitas visual, dan pengalaman naratif yang telah didefinisikan pada DESIGN.md dan PRD.md.

---

# Implementation Strategy

## Core Strategy

Implementasi dilakukan secara bertahap.

Setiap fase harus selesai dan tervalidasi sebelum melanjutkan ke fase berikutnya.

## Guiding Principles

* Pertahankan DESIGN.md.
* Ikuti PRD.md.
* Patuhi INSTRUCTION.md.
* Jangan melakukan redesign.
* Jangan membuat asumsi baru.
* Bangun fondasi terlebih dahulu sebelum animasi.

## Development Philosophy

Build stable first.

Build beautiful second.

Build animation last.

# Definition of Done

Website dianggap selesai apabila:

* Sesuai DESIGN.md.
* Sesuai PRD.md.
* Sesuai INSTRUCTION.md.
* Storytelling tetap utuh.
* Journey tetap terasa.
* Identitas visual tidak berubah.
* Responsive.
* Accessible.
* Optimized.
* Maintainable.
* Tidak ada fitur tambahan yang tidak terdokumentasi.

# Final Principle

DESIGN.md menjelaskan bagaimana website terlihat.

PRD.md menjelaskan mengapa website dibuat.

INSTRUCTION.md menjelaskan bagaimana AI harus membangunnya.

IMPLEMENTATION_PLAN.md menjelaskan urutan implementasi yang harus diikuti.

Jika terjadi keraguan:

STOP.

Kembali ke DESIGN.md.

Jangan membuat asumsi baru.
