# Hasmunandar Portfolio Website

Website e-portofolio pribadi berbasis **storytelling** yang dirancang untuk memperkenalkan identitas, perjalanan pendidikan, kemampuan, dan karya digital dari **Hasmunandar** (Mahasiswa PPG Prajabatan & PGSD, Calon Guru Profesional, dan Pendidik Sekolah Dasar).

## Teknologi yang Digunakan

- **Framework Utama**: Next.js (App Router) & React
- **Bahasa Pemrograman**: TypeScript
- **Styling**: Tailwind CSS v4 (dengan variabel tema kustom di `globals.css`)
- **Library Animasi**: GSAP & Framer Motion
- **Utilitas**: `clsx` & `tailwind-merge` (dikelola lewat `@/lib/utils`)
