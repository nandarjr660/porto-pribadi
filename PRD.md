# Product Overview

## Product Name

Hasmunandar Portfolio Website

## Product Summary

Hasmunandar Portfolio Website adalah website personal portfolio berbasis storytelling yang dirancang untuk memperkenalkan identitas, perjalanan pendidikan, kemampuan, dan karya pemilik portfolio kepada recruiter, institusi pendidikan, rekan profesional, serta calon kolaborator.

Website ini tidak hanya berfungsi sebagai showcase karya, tetapi juga sebagai representasi perjalanan pribadi dan profesional yang terus berkembang dari waktu ke waktu.

## Product Purpose

Produk ini dibuat untuk:

* Memperkuat personal branding.
* Menampilkan identitas profesional pemilik portfolio.
* Mendukung kebutuhan rekrutmen.
* Membuka peluang kolaborasi.
* Menjadi pusat dokumentasi perjalanan karier dan pendidikan.
* Menjadi media promosi kemampuan pembuatan website dan proyek digital.

## Core Value Proposition

### Storytelling First

Pengunjung memahami perjalanan sebelum melihat hasil.

### Human-Centered Identity

Portfolio memperkenalkan manusia di balik karya.

### Evidence Through Work

Kemampuan dibuktikan melalui karya nyata.

### Long-Term Growth

Portfolio terus berkembang mengikuti perjalanan karier dan pendidikan.

# Problem Statement

## Existing Problem

Sebagian besar portfolio hanya menampilkan daftar proyek atau pencapaian tanpa memberikan konteks mengenai perjalanan yang membentuk pencapaian tersebut.

CV juga sering kali tidak mampu menjelaskan proses pertumbuhan, karakter, dan identitas profesional secara menyeluruh.

## User Needs

Pengunjung membutuhkan media yang dapat membantu mereka:

* Mengenal pemilik portfolio.
* Memahami latar belakang pendidikan.
* Memahami kemampuan yang dimiliki.
* Melihat hasil nyata dari kemampuan tersebut.
* Menjalin komunikasi atau kolaborasi.

# Features

## Preloader

### Purpose

Menyambut pengunjung dan memperkenalkan tema perjalanan.

### Behavior

* Menampilkan progress loading.
* Menampilkan pesan sambutan.
* Menjadi pembuka cerita sebelum website ditampilkan.

### User Value

Memberikan konteks awal mengenai konsep perjalanan yang menjadi tema utama website.

---

## Navigation Overlay

### Purpose

Menyediakan navigasi utama sekaligus memperkuat pengalaman storytelling.

### Behavior

* Membuka navigasi penuh layar.
* Menampilkan identitas pemilik.
* Memberikan transisi antar halaman yang lebih terarah.

### User Value

Membantu navigasi tanpa menghilangkan pengalaman editorial.

---

## Roadmap Timeline

### Purpose

Menyampaikan perjalanan pendidikan secara kronologis.

### Behavior

* Menampilkan perjalanan bertahap.
* Mengungkap milestone pendidikan secara berurutan.

### User Value

Membantu memahami proses yang membentuk kemampuan dan identitas profesional.

---

## Skills Accordion

### Purpose

Mengelompokkan kemampuan agar mudah dipahami.

### Behavior

* Membuka dan menutup kategori kemampuan.
* Menampilkan satu kategori aktif pada satu waktu.

### User Value

Menyediakan informasi yang terstruktur dan mudah dieksplorasi.

---

## Project Showcase

### Purpose

Menampilkan hasil nyata dari kemampuan yang dimiliki.

### Behavior

* Menampilkan daftar project.
* Mengarahkan pengguna ke website project.

### User Value

Memungkinkan pengguna memverifikasi kualitas dan hasil kerja secara langsung.

---

## Contact Links

### Purpose

Menyediakan jalur komunikasi yang mudah.

### Behavior

* Menghubungkan pengguna ke email.
* Menghubungkan pengguna ke media sosial.

### User Value

---

## Footer

### Purpose

Menjaga konsistensi navigasi dan identitas website.

### Behavior

* Tersedia pada seluruh halaman.
* Menjadi penutup setiap bagian pengalaman.

### User Value

Memberikan pengalaman yang konsisten dan terstruktur.
