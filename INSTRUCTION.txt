# INSTRUCTION.md

# Project Context

## Project Summary

Website ini adalah personal portfolio berbasis storytelling yang memperkenalkan identitas, perjalanan pendidikan, kemampuan, dan karya pemilik portfolio.

Website bukan sekadar project showcase, tetapi pengalaman naratif yang membawa pengunjung memahami proses yang membentuk pencapaian tersebut.

## Core Philosophy

* Storytelling
* Journey
* Simplicity
* Human-centered experience

# Animation Rules

## Core Principle

Motion guides the story.

## Motion Characteristics

* Smooth
* Calm
* Cinematic
* Editorial
* Human

## Motion Objective

Animasi digunakan untuk:

* Mengungkap cerita
* Membimbing pengguna
* Membangun ritme membaca

Animasi tidak digunakan untuk memamerkan efek visual.

## Rule

Reveal, not celebrate.

## Avoid

* Flashy animation
* Aggressive transition
* Unnecessary effects
* Over animation

---

# Responsive Rules

## Allowed Adaptation

AI boleh menyesuaikan:

* Layout position
* Element stacking
* Element sizing
* Reading flow

untuk tablet dan mobile.

---

## Not Allowed

AI tidak boleh mengubah:

* Storytelling structure
* Color palette
* Typography identity
* Main hierarchy
* Design philosophy

---

## Responsive Goal

Menjaga identitas visual pada seluruh ukuran layar.

---

# Accessibility Rules

## Semantic Structure

Gunakan semantic HTML.

## Keyboard Navigation

Seluruh interaksi harus dapat diakses dengan keyboard.

## Color Contrast

Pertahankan keterbacaan yang baik.

## Reduced Motion

Hormati preferensi pengguna terhadap reduced motion.

## Focus Visibility

Pastikan focus state tetap jelas.

# Performance Rules

## Lazy Loading

## Animation Optimization
